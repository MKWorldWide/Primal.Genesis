# Override Core

Node.js development override and file watching system.

## Purpose

Provides development tooling for:
- File system monitoring and overrides
- Development workflow automation
- Hot reloading and live updates
- Development environment management

## Current State

**Phase 2: Skeleton Created**

Functional Node.js application exists in `override_core/` directory and will be migrated here.

## Features

- File watching with chokidar
- Express server for development
- Override rule processing
- Log watching and monitoring

## Migration Plan

Current `override_core/` directory will be moved to `apps/override-core/` with:
- Updated package.json dependencies
- Improved TypeScript support
- Enhanced configuration management
- Better error handling and logging
