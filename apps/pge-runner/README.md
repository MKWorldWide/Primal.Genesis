# PGE Runner

TypeScript policy engine and watcher system.

## Purpose

Provides policy execution and monitoring for:
- Policy rule processing
- File system watching
- Event-driven architecture
- Runtime policy enforcement

## Current State

**Phase 2: Skeleton Created**

Functional TypeScript policy engine exists in `pge/` directory and will be migrated here.

## Features

- Policy engine with watcher primitives
- Event-driven rule processing
- Configuration-based policy management
- Runtime monitoring and logging

## Policy Primitives

Current watcher primitives include:
- `black-sun-core.absorb` - Resource absorption
- `serpentine.transmute` - State transformation
- `lunar-mirror.scrub` - Data cleaning

- `solar.assert` - Sovereign assertion
- `crown.route` - Request routing
